# Dự án phân loại Coca (Mở / Đóng)

## Mô tả
Đây là một dự án đơn giản về thị giác máy tính, dùng để phân loại lon Coca-Cola thành 2 trạng thái:

- 🟢 Lon đã mở
- 🔴 Lon chưa mở (đóng)

Hệ thống được thiết kế mô phỏng cho dây chuyền sản xuất, trong đó sử dụng camera điện thoại để chụp hình và chương trình sẽ tự động phân loại trạng thái sản phẩm.

>  Phiên bản hiện tại chỉ hỗ trợ phân loại 2 trạng thái: MỞ và ĐÓNG.

---

##  Mục tiêu dự án
- Hỗ trợ tự động hóa trong dây chuyền sản xuất
- Sử dụng camera điện thoại để lấy hình ảnh
- Phân loại trạng thái sản phẩm (mở / đóng)
- Là nền tảng để phát triển hệ thống AI kiểm tra chất lượng sau này

---

##  Chức năng
- Phân loại hình ảnh đơn giản
- Quản lý dữ liệu trong thư mục `Dataset/`
- Chạy bằng Python qua file `main.py`
- Có thể mở rộng sang AI / Machine Learning
